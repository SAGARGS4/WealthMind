# WealthMind – AI-Powered Personal Finance Platform

## Overview
WealthMind is a full-stack AI-powered application that helps users track expenses, analyze spending patterns, and manage personal finances efficiently.

## Features
- User authentication
- AI-based expense categorization
- Real-time dashboard and analytics
- Budget alerts
- Secure data storage

## Tech Stack
- Frontend: React.js, Next.js
- Backend & Database: Supabase (PostgreSQL)
- AI Integration: Expense categorization
- Visualization: Charts and analytics

## Architecture
User → React/Next.js → API Routes → Supabase → AI → Dashboard

## Setup
1. Clone the repository
2. Install dependencies
3. Configure Supabase keys
4. Run the development server

## Future Scope
- Bank API integration
- Advanced AI insights
- Mobile app support
